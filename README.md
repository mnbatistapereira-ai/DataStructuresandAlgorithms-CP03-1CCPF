Descrição do sistema

Sistema simples de caixa eletrônico desenvolvido em linguagem C. O programa permite consultar saldo, realizar saques e depósitos através de um menu interativo no terminal.

Funcionalidades implementadas:
Consulta de saldo;
Realização de saque;
Realização de depósito;
Validação de saldo insuficiente;
Validação de valores inválidos;
Limite diário de saque;
Menu interativo com repetição até o usuário sair.

Instruções de compilação e execução
gcc main.c -o main
